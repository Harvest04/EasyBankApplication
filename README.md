# EasyBankApplication

EasyBankApplication is a Java console-based banking system that demonstrates control flow conditions and basic object-oriented programming concepts.

The application allows users to:
- Check account balance
- Deposit money
- Withdraw money
- Buy airtime
- Buy electricity (PPA simulation)

User interaction is handled using control flow conditions such as `if`, `else`, `switch`, and loops to manage menu navigation and validate transactions.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Control Flow Conditions
- Packages and Classes

## How to Run
1. Open the project in any Java IDE (IntelliJ, NetBeans, Eclipse)
2. Compile and run `MainApp.java`
3. Follow the on-screen menu
